# Atlassian Cursor Plugin

A dedicated [Atlassian](https://atlassian.com) plugin that bridges your AI assistant with the full suite of Atlassian cloud products.

## Overview

This plugin connects your AI assistant to Atlassian.

## Installation

1. Open Cursor settings
2. Navigate to **Plugins**
3. Click **Browse Marketplace**
4. Search for "Atlassian"
5. Click **Install**
